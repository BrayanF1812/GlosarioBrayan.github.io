A continuación encontrarás las definiciones de las principales palabras y expresiones relacionadas con la asignatura.

ACESS POINT
el punto de acceso es un hardware que se añade a un router el cual nos permite aumentar el campo de concectividad haciauna red, generalmente wifi,


AM
Amplitud modulada


ANCHO DE BANDA
Capacidad de transmicion de datos


ANTENA PARABOLICA
la antena parabolica se encarga de recivir las señales que envia un satelite, cabe reacalcar que al ser parabolica sea donde sea que lleguen las señales se veran reflejadas hacia el foco de esta antena


ARPANET

Red Militar



BOCINA
es un dispositivo capaz de trasformar la energía eléctrica en energía acústica


CABLE COACCIAL
es un medio de transmicion alambrico , esta compuest opor dos hilos de cobre uno para enviar y otro para recibir, tambien cada hilo esta separado por un recubrimiento con el fin de evitar la impedancia


CABLE UTP
el cable utp o tambien conocido como ethernet es la manera alambrica de que nuestro router le entregue internet a nuestra computador, pese a ser algo antiguo es la que mas se usa actualmente, cabe aclarar que este tiene en su interior cuatro pares de cables, consta de 8 pines


CYCLADES
Red cientifica en Francia


DARPA

Ajencia para el projecto de investigacion en la Agencia



DNS
Servidor de nombres de dominio


EMISOR
Aquel que envia un mensajes


ESPECTRO ELECTRONMAGNETICO
es la distribución de energías de las radiaciones electromagnéticas.


FM
Frecuencia modulada


FIBRA OPTICA
La fibra optica es un medio de transmicion que usas los aces de luz oara la transmicion de datos


FIREWALL
Tambien conocido como cortafuegos es un sistema de seguridad qur protege las redes informaticas, su funcion es controlar el trafico que entra y sale baasandose en un conjuntos de reglas de seguridad


HTTP
protocolo de transferencia de hipertexto


HZ
unidad de media del sonido


IMPEDANCIA
la impedancia es un fenomeno de oposicon de las cargas al viajar mediante hilos de cobre,se puede evitar mediante diferentes metodos como lo hace el cable coaccial o como lo hace el par trensado


INTERFAZ
La interfaz es un tipo de menu que funciona como comunicador entre el usuario y la maquina, este permite que la maquina y el usuario se puedan comunicar


IP
internet Protocol direccion asociada a un equipo o pagina web conformada por una serie de numeros


ISO
Organizacion Internacional de Normalizacion


MAC
ALa direccion mac es una direccion quwe identifica nuestro equipo de manera global, esta incluye numeros, el modelo y empresa de nuestro equipo


MEDIO DE TRANSMICION
componente fisico a traves el cual viaja la inormacion


MODEM
Convierte los atos en señales electricas, electromagneticos o de haces de luz


MODELO DE RED
Representacion del funcionamiento de una red. El modelo no es la red real


MODULADOR
el modulador es un dispositivo encargado de convertir las ondas electromagneticas en datos


MULTIPLEXACION
La multiplexacion es el proceso de convertir datos a ondas y viceversa


NPL
Red comecial en laboratorio naciona de fisica en Inglaterra


LAN
local area network es una red local, podemos tomar como ejemplo una red de computadores y celulares conectados entre si junto a un router, esto lo podemos considerar como una LAN


PROTOCOLO
Pasos / Reglas a seguir o cumplir


PSI
Proveedores de servicios de internet


QoS
Calidad de servicio (Quality of Service) se refiere a la capcidad de ofrecer diferentes niveles de prioridad a las diferentes aplicaciones y personas


RECEPTOR
Aquel que recibe el mensaje


SMTP
Protocolo usado para emison y recepcion de correos electronicos


TELETRABAJADORES
aquellos individuos que trabajan a distancia


TOPOLOGIA DE RED
Como estan conectados los diferentes nodos


URL
Localizador de recursos uniforme


WI-FI
wireless fidelity o tambien conocidad como wifi hace referencia a una red de alta fidelidad que funciona mediante ondas


WPAN
WPAN hace referencia a una red persona inalambrica, podemos tener como ejemplo bluetooht
