A continuación encontrarás las definiciones de las principales palabras y expresiones relacionadas con la asignatura.

A

ACESS POINT
el accesss point o en español punto de acceso es un hardware que se añade a un router el cual nos permite aumentar el campo de concectividad de una red wifi permitiendo asi que si nos distanciamos del modem pero estamos cerca de el access point se mantenga la coneccion permitiendonos cumplir nuestros deberes o necesidades


AM
la amplitud modulada se encarga de controlar la disposicion de espectro electromagnetico, esta fue una funcion vital debido a que nos permite que muchos emisores emitan a diferentes amplitudes permitiendo que el receptor pueda clasificar estas ondas para escuchar aquellas que el desea


ANCHO DE BANDA
el ancho de badna viene siendoCapacidad de transmicion de datos, esta es proporcional a su nivel, para hacerlo entender mejro entre mas acnho de banda presente mas datos sera capaz de enviar simultaneamente


ANTENA PARABOLICA
la antena parabolica se encarga de recivir las señales que envia un satelite, cabe reacalcar que al ser parabolica sea donde sea que lleguen las señales se veran reflejadas hacia el foco de esta antena


ARP

protocolo de resolucion de direcciones es el encargado de encontrar la direccion de un hardware, una direccion IP



ARPANET

ARPANET fue una red de computadores militar creada por el departamento de defensa de los estados unidos



ASCII

ASCII es un codigo de caracteres basado en el alfabeto latino



B

BOCINA
es un dispositivo capaz de trasformar la energía eléctrica en energía acústica


C

CABLEADO ESTRUCTURADO
el cableado estructurado hace refencia a las conecciones de los cables de par trenzado conectados a traves de un edificio con el fin de montar una red LAN


CABLE COACCIAL
es un medio de transmicion alambrico , esta compuest opor dos hilos de cobre uno para enviar y otro para recibir, tambien cada hilo esta separado por un recubrimiento con el fin de evitar la impedancia


CABLE UTP
el cable utp o tambien conocido como ethernet es la manera alambrica de que nuestro router le entregue internet a nuestra computador, pese a ser algo antiguo es la que mas se usa actualmente, cabe aclarar que este tiene en su interior cuatro pares de cables, consta de 8 pines


CABLE VGA
El cable vga es caracterizado por presentar un color azul en sus conectores, este se encarga de la salida exclusivamente de la imagen de la computadora


CANALETA
la canaleta es una herramienta muy utilizada en las redes debido a que nos permite llevar cables en medio de esta, al ser una forma por lo generalmente geometrica y regular nos permite tener mayor orden en cables de oficina y tambien brindarles aun mas proteccion


CAPUCHON
se le denomina capuchon a una especie de caucho adaptado a la forma de un conector rj45 macho con el fin de poder aumentar la proteccion y vida util de este mismo


CIFRADO
el cifrado es un metodo de codificacion de datos que impide que puedan ser legidos a exepcion de los si autorizados. esto se hace usadno una clavee criptografica


CIFRADO DEL CESAR
Es un proceso de codificacion que busca cambiar las letras que se usan en la carta con el fin de que sea mas complejo su entendimiento para aquellos que intente leerla y no sean el usuario destino


CODIFICACION
La codificacion es el proceso de almacenar y recuperar informacion


COMUNICACION
Es la interaccion entre dos personas ya sea mediante palabra verbal o escrita, señas, etc


CRYMPADORA
herramienta que nos permite asegurar nuestros rj45 a los cables utp, se encarga de presar las pestañar interiores del rj45 al cable utp permitiendo el correcto funcionamiento de este y que cuando se este usando el rj45 no se "caiga" solo del cable


CYCLADES
Red cientifica en Francia


D

DARPA

Ajencia para el projecto de investigacion en la Agencia



DATA CENTER

un data center o en español centro de procesamiento de datos hace referencia a una sala usada para el almacenamietno de diferentes equipos encargados del flujo de los datos,ademas de esto es capaz de proporcionar seguridad a estos datos.



DECIFRADO

Es el proceso en que el que al recibir el mensaje ya codificado se vuelve a su estado inicial para poder ser entendido



DHCP

El tambien conocido como Dynamic Host Configuration Protocol es un protocolo que funciona en nuestros routers que se encarga de asignarnos automaticamente una ip a nuestros dispostivos con el fin de ahorrarnos el profeso de hacerlo manual y de paso tener acceso a internet lo mas rapido posible, tambien podemos aclarar que como es dinamico las ip no seran siempre las mismas, estas pueden llegar a variar.



DIRECCION
La direccion es una ubicacion fisica en concreto nombrada con cierta serie ya sea de numeros caracteres o los dos juntos ,es muy util en redes ya que nos permite enviar el archivo exactamente a donde nosotros queremos


DNS
Servidor de nombres de dominio


E

EMAIL
Un email es un correo electronico, nos permite enviar y recibir diferente tipo de informacion y ademas de esto podemos enviarlo a multiples usuarios a la vez


EMISOR

ESPECTRO ELECTRONMAGNETICO

ETHERNET
El ethernet es un protocolo fisico del internet el cual nos permite la coneccion de diferentes equipos ya sea de manera personal como en la casa, o de manera a gran escala como podria ser una empresa


F

FACE PLATE
el face plate es similar a una toma corriente pero se le cambian sus puertos a jacks rj45, nos permite tener una conexion del modem a algun lugar en espefico, bien puede ser oficinas, cuartos etc, en donde podemos conectar nuestro equipo con un cable utp a este face plate y desconectarlo cuando queramos sin necesidad de mover el cable que se dirije al face plate

Frecuencia modulada

La fibra optica es un medio de transmicion que usas los aces de luz oara la transmicion de datos

HZ
unidad de media del sonido


HOST
el host o tambien conocdio como hosting hace referencia a una computadora o equipo conectada a una red mediante un numero de ip definitdo. Ademas de esto es capaz de brindar informacion o recursos a sus ususarios


I

IMAP
Es un protocolo de aplicacion que permite el acceso a los mensajes de un servidor


IMPEDANCIA
la impedancia es un fenomeno de oposicon de las cargas al viajar mediante hilos de cobre,se puede evitar mediante diferentes metodos como lo hace el cable coaccial o como lo hace el par trensado


INFRAESTRUCTURA

MEDIDOR DE CABLE
es un intrumento que nos facilita la medicion del cable a comparacion de otros instrumentos de medicion como lo son el metro. la ventaja de este es que tiene una rueda la cual cuenta con un canal en donde se acomoda el cable y mediante el recorrido de esta rueda por todo el cable podemos conocer la longitud de este mismo

MULTITOMA
